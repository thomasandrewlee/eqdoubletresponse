The script GetInstrumentResponseWithDoublets.jl is a julia script which reads data from earthquake doublets recorded on both modern and historical seismic data sources, performs spectral division with a correction for microseism, and determines an empirical response.

The text files "ISC_M5.5..." contain the earthquakes considered (from the ISC catalog) to find doublet pairs.

Additionally, the set of plots in the style of Figure 3 in Lee et al. for all events may be found here: https://drive.google.com/file/d/1vb61Ho1eq02KgJWRvceVhjDjYAkSemba/view?usp=sharing
